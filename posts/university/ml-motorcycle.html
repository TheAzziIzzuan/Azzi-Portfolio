Machine Learning Project

Motorcycle Price Prediction for the Singapore Market

Overview

A full-stack machine learning web application that predicts the resale price of motorcycles in Singapore. The system trains and compares five different ML algorithms, then lets users pick the best-performing model through an admin panel. Regular users simply enter their bike's details — engine size, mileage, COE expiry, number of owners — and get an instant price estimate.

Singapore's motorcycle market is unique because of the Certificate of Entitlement (COE) system, which heavily influences vehicle pricing. The models account for this by treating COE expiry as a key feature, adjusting predictions based on how many years of COE are left on the vehicle.

Models Trained & Compared

Five regression models were trained on the same dataset and evaluated side-by-side using MAE, RMSE, and R² score:

Random Forest

Ensemble of decision trees that averages predictions to reduce overfitting. Good baseline model that handles non-linear relationships and mixed feature types without heavy preprocessing.

XGBoost

Gradient-boosted decision trees with regularisation. Builds trees sequentially, where each new tree corrects the errors of the previous ones. Strong performance on structured tabular data.

LightGBM

Microsoft's gradient boosting framework using leaf-wise tree growth for faster training. Includes post-prediction adjustments for engine capacity, owner count, and mileage to fine-tune estimates.

Support Vector Machine (SVM)

SVR with RBF kernel, trained using a pipeline with RobustScaler for outlier-resistant normalisation. Hyperparameters tuned via RandomizedSearchCV. Log transformation applied to target prices for better distribution handling.

CatBoost

Handles categorical features natively without explicit encoding. Uses advanced feature engineering including log-transformed mileage and engine capacity, polynomial interactions, and bike age derived features.

Input Features

Engine Capacity (cc) Displacement of the motorcycle engine — the single strongest price signal in the dataset
Registration Date Year the bike was first registered in Singapore, used to calculate vehicle age
COE Expiry Date When the Certificate of Entitlement expires — more years left means significantly higher value
Mileage (km) Total distance ridden; higher mileage reduces predicted price through depreciation modelling
Number of Owners Previous ownership count — each additional owner applies a ~10% depreciation factor
Brand & Category Manufacturer (Honda, Yamaha, Ducati, BMW, etc.) and bike type (Sport, Cruiser, Scooter, Touring, etc.) — label-encoded for tree models, one-hot encoded for SVM

Data Pipeline

  • 1 — Data Collection Motorcycle listing data scraped and compiled from the Singapore resale market, stored in combined_dataset_latest.xlsx
  • 2 — Cleaning & Preprocessing Strip currency symbols and non-numeric characters from price, engine capacity, and mileage columns. Handle missing values with median imputation. Remove statistical outliers using z-score filtering.
  • 3 — Feature Engineering Derive Bike_Age from registration year, COE Years Left from expiry date, log-transform skewed features (mileage, engine capacity), and create interaction terms (Engine_Capacity × Bike_Age, Mileage × COE_Years_Left)
  • 4 — Encoding & Scaling Label encoding for categorical features (Brand, Category). RobustScaler for SVM to handle outliers. One-hot encoding variant used by SVM to produce 52-feature vectors.
  • 5 — Training & Evaluation 80/20 train-test split. Each model trained independently. Performance compared on MAE, RMSE, and R². Hyperparameter tuning via RandomizedSearchCV for SVM and CatBoost.

Web Application

The Flask web interface has two roles. Users fill in a form with their motorcycle's details and get an instant price prediction powered by the currently active model. Admins can view all five models' performance metrics side-by-side, switch the active model, toggle input filters (license class, mileage range, COE years left), and access visualisations — scatter plots of actual vs. predicted prices, error distributions, residual plots, and feature importance charts.
