Digital Forensics

The purpose of this assignment is to research on the tools that can carry out forensic analysis in a professional and efficient manner. The use of dcfldd will help to verify the integrity of disk imaging or wiping processes and gather important information from disk images, files, or collections of files. Additionally, bulk-extractor provides the capability to uncover hidden information, such as email addresses, URLs, and credit card numbers, from compressed data and fragmented or partially damaged files. With these tools at hand, the aim of the assignment is to gather information and evidence in an organized and thorough manner. The assignment was successfully completed by Rafael and Azzi.

First tool - dcfldd 1.3.4-1 (by Rafael Kamil)

dcfldd is a sophisticated iteration of the popular GNU dd software, designed specifically for forensic and security purposes. Building upon the foundation of the dd program found in the GNU Coreutils package, dcfldd offers enhanced capabilities, including the following additional features:

  • - dcfldd provides status updates to keep the user informed on the progress of data transfer. These updates include the amount of data transferred and an estimate of how much time remains until the operation is complete.

  • - dcfldd offers versatile disk wiping capabilities, allowing for the efficient and thorough elimination of data, with the option to use a desired pattern if desired.

  • - dcfldd provides the ability to verify the accuracy of disk imaging or wiping processes, ensuring that the target drive is an exact replica of the specified input file or pattern, down to each individual bit.

Second tool - Bulk-Extractor (by Azzi Izzuan)

bulk_extractor is a powerful C++ tool that enables the extraction of valuable information from disk images, files, or collections of files, without the need for parsing the underlying file system or its structures, including the following additional features:

  • - Upon completion of the analysis, create a histogram that visually represents the distribution of information, including email addresses, URLs, domain names, search keywords, and other relevant data.

  • - Uncover previously undiscovered information, including email addresses, URLs, and credit card numbers, using a tool that can efficiently handle compressed data like ZIP, PDF, and GZIP files, as well as fragmented or partially damaged data. This tool has the ability to extract JPEG files, office documents, and various other file types from within compressed data, and can seamlessly detect and extract encrypted RAR files.

  • - Construct a comprehensive list of words discovered within the data, including those found in compressed files that lack allocated space. This word list can be utilized in password cracking efforts.
